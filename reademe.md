# 웹퍼블리셔 지원자 - 신성호

## 프로젝트 개요
컴투스의 이벤트 페이지를 퍼블리싱한 작업입니다.  
디자인 시안에 따라 마크업을 구현하고, 스크립트를 통해 인터랙션 효과를 구현했습니다.

### 주요 작업 내역
- HTML5 / CSS3 / JavaScript 퍼블리싱
- 스탬프 및 게이지 활성화 효과 (`.on` 클래스 활용)
- 시청 횟수가 클릭에 따라 증가하는 인터랙션 구현
- keyframes을 사용해 팽귄 이미지가 자연스럽게 이동하는 효과 구현
